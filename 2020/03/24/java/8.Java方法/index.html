Java方法

Java中方法的定义,参数,返回值等;

Java方法

用于解决一类问题的代码的有序组合,是一个功能模块。其主要功能是用来复用。

语法格式

调用

(new 对象).方法名(参数如果有)

参数的传递

  • 值传递(可以加&符变为引用传递)
  • 引用传递

可变参数

方法的重载

方法名相同,参数不同(参数个数不同或者方法个数相同,但类型不同)

方法之间的调用

  • 非静态方法可以调用静态和静态的方法
  • 静态方法只能调用静态的方法
