PHP运算符

PHP中运算符的分类及其优先级.

表达式与语句

  • 通过一定的运算符连接并且有值的式子
  • 语句通常值表达式后加分号

注: 表达式不等于语句

运算符的定义

连接两个或多个变量并进行计算的符号。

运算符的分类

  • 算术运算符
  • 递增递减运算符
  • 字符连接符
  • 赋值运算符
  • 比较运算符
  • 逻辑运算符
  • 错误抑制符
  • 数组运算符(与算术和比较运算符有重叠)
  • 三元运算符
  • 组合比较符(PHP7,其实是比较运算后的赋值)
