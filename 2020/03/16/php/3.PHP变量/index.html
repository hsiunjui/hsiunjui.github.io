变量定义

变量是用于存储信息的容器

PHP变量的命名规则

  • 变量必须以$打头,后面紧跟变量名称
  • 变量名称必须是以字母或者是_(下划线字符)开始
  • 变量名称只能包含数字字母下划线[A-z0-9_]
  • 变量名不能包含空格
  • 变量名区分大小写
  • 注意：

变量的作用域

  • global(全局作用域)
  • local(局部作用域)
  • static(静态作用域)
  • parameter(参数作用域)
