系统函数库

PHP内置的一些全局函数。

函数分类

  • 字符函数(见数据类型篇)
  • 数学函数(同上)
  • 数组函数(见数组篇)
  • 类型检测函数(同上)
  • 输出函数
  • 时间日期函数
  • 文件操作函数(见文件操作篇)
  • 错误处理函数(见错误级别篇)
  • 序列化
    serialize($var); // 产生一个可存储值的表示,返回字符串,此字符串包含了$var的字节流,可以存储在任何地方
unserilize(); // 反序列化
json_encode(); // 对变量进行json编码,返回字符串
json_decode(); // 对json格式的字符串进行解码
  • 编码
    base64_encode(); // 使用MIME base64对数据进行编码
base64_decode(); // 对使用MIME base64编码的数据解码
