PHP操作数据库(三) - PDO

PHP使用PDO方式对数据库的增删改查操作.

PDO

能实现MySQLi的常用功能,支持大部分数据库.

步骤

1.连接数据库

2.执行SQL语句

  • PDO::exec($sql); 用于没有记录返回的,如UPDATE,INSERT,返回受影响的行数
  • PDO::query($sql); 用与有记录返回的,如SELECT,获取结果集
    • PDOStatement::fetch(); 获取一条记录
    • PDOStatement::fetchAll(); 获取所有记录集合
