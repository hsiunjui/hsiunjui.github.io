PHP邮件发送

使用PHP进行邮件发送.

邮件的应用场景

  • 身份验证
  • 市场营销
  • 邮件报警

SMTP协议

SMTP(Simple Mail Transfer Protocol),即简单邮件传输协议,它是一组用于由源地址到目的地址传送邮件的规则,由它来控制信件的中转方式。

发送邮件的方式

  • 使用telnet命令
  • 使用mail命令
  • 使用代码实现SMTP协议
  • 使用php提供的mail函数
  • 使用第三方库
    • swiftmailer
    • phpmailer
  • 使用第三方服务
    • Mailgun
    • AWS SES
    • QQ企业邮箱

  • Redis
  • beanstalk
  • rabbitMQ
