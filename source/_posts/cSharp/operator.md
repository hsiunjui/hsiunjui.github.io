>c#语言中的运算符种类与规则。

<!--more-->
# 一元运算符
    1. ++
    2. --
        注：前自增（减）与后自（减）的区别
            一元运算符优先级较高

# 算术运算符
    + - * / %

# 关系运算符
    1. > < =
    2. >= <= == !=

# 逻辑运算符
    1. &&
    2. ||
    3. !
        注：逻辑运算符两边一般都关系表达式

# 符合运算符
    += -= *= /= %=

# 三元运算符
    ? :

# 位运算符
    & | >>>
